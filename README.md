# Employee Management System

## Overview

A Full Stack Employee Management System built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). This application allows administrators to manage employee records with secure authentication and an interactive dashboard.

## Features

* User Registration & Login
* JWT Authentication
* Protected Routes
* Add Employee
* Edit Employee
* Delete Employee
* Search Employee
* Dashboard with Statistics
* Department-wise Charts
* Responsive Sidebar
* MongoDB Database Integration

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Axios
* React Router DOM
* Recharts

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT
* bcrypt

## Installation

### Clone Repository

```bash
git clone https://github.com/rutujagiribuva/Employee-Management-System.git
```

### Install Dependencies

Client

```bash
cd client
npm install
npm run dev
```

Server

```bash
cd server
npm install
npm start
```
